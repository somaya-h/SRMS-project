

Student Record Management System (SRMS)


This file provides instructions for compiling, running, and testing
the Student Record Management System (SRMS) from the command line.

-------------------------------------
Requirements
- Java JDK (version 8 or later)
- Command Line / Terminal
- All project files in the same directory

-------------------------------------
How to Compile
1. Open Command Prompt (or Terminal).
2. Navigate to the project directory.
3. Compile all Java files using the following command:

   javac *.java

-------------------------------------
How to Run
After successful compilation, run the program using:

   java SRMSApp

-------------------------------------
How to Use the System
- Follow the menu displayed on the screen.
- You can add, update, remove, search, and display student records.
- Data is automatically saved using the AutoSaver feature.

-------------------------------------
Notes
- Make sure all .java files are in the same folder.
- If an error occurs, check that Java is properly installed
